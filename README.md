# 📊 Student Marks Analyzer

This is a Python-based project that analyzes student marks and provides meaningful insights such as total marks, average score, grade, and top-performing student.

---

## 🚀 Features
- Calculates total and average marks of each student
- Assigns grades based on performance
- Identifies the top-performing student
- **Accepts user input for real-time analysis**

---

## 🛠️ Technologies Used
- Python

---

## 📌 How It Works
The program asks the user to enter student names and their marks. It then calculates total, average, and grade for each student and identifies the top student.

This project demonstrates basic data analysis, problem-solving, and Python programming skills.
